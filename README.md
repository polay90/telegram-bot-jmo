# 🤖 Bot Telegram JMO Troubleshooting

Bot Telegram yang dirancang untuk membantu pengguna dengan masalah JMO Jamsostek.

## 📋 Fitur Utama

✅ **Login dengan WhatsApp**
- Verifikasi nomor WhatsApp
- Kode OTP 4 digit

✅ **Menu Layanan**
- 🔧 Kode 040 - Solusi error kode 040
- 🔐 Suspend Nomor KPJ - Buka suspend KPJ
- 🔑 Unlock Biometrik - Unlock biometrik Jamsostek

✅ **Fitur Tambahan**
- Progress bar real-time
- Pengiriman data otomatis ke admin
- Interface yang menarik dengan emoji
- Penyimpanan data user dalam format JSON
